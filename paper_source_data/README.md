# Paper Source Data

This directory contains the small CSV files used to generate or audit the manuscript figures and numerical summaries for the TopoIWL-Net Remote Sensing submission.

The files are derived from the completed experiment outputs and manuscript figure-preparation workflow. They do not contain original remote sensing images, dataset archives, model checkpoints, or prediction rasters.

Included files:

- `figure4_dataset_overview_source_data.csv`: dataset split and composition summary.
- `figure5_region_boundary_metric_gap_source_data.csv`: GF6 and SeaLand main comparison values.
- `figure6_accuracy_efficiency_tradeoff_source_data.csv`: accuracy-efficiency source values across datasets.
- `figure7_ablation_contribution_source_data.csv`: GF6 and SeaLand ablation source values.
- `figure_s3_selected_thresholds_source_data.csv`: validation-selected thresholds.
- `figure_s3_threshold_radius_sensitivity_source_data.csv`: threshold/radius sensitivity source values.
- `figure8_qualitative_manifest.csv`: qualitative comparison manifest.
- `figure_s1_extended_qualitative_manifest.csv`: supplementary qualitative manifest.
- `figure_s2_label_products_manifest.csv`: supplementary label-product manifest.

Users must obtain the original datasets from their respective providers. This repository intentionally excludes restricted imagery and large experiment artifacts.
